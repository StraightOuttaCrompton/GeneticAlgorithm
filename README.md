Step 1: Initialise - create a population of N elements, each with randomly generated DNA

DRAW
Step 2: Selection: Evaluate the fitness of each element of the population and build a mating pool

Step 3: Reproduction: Repeat N times
    a. Pick two parents with a probability according to fitness
    b. Breed parents to create child
    c. Mutate child
    d. add child to population

Step 4: replace old population with new population and return to Step 2
